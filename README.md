## Deployment

The project is deployed at netlify

```bash
  https://wavy-rsvp.netlify.app/
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/tusharbansal22/Wavy-frontend
```

Go to the project directory

```bash
  cd Wavy-frontend
```

Install dependencies

```bash
  npm i
```

Start the client

```bash
  npm start
```
